# Frontend - Notes App

## Requirements

- Node.js v18.x or superior
- npm v9.x o superior

## Install and run

```bash
npm install
npm start
```

Frontend runs on http://localhost:5173 and connects to the backend at http://localhost:4000

## Tech stack

- React (Vite)
- Axios

No environment variables needed.

## UI

SPA for notes creation, editing, archiving and deleting, with a clean and modern look.
